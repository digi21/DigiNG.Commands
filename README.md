* copiar_atributos_bbdd
* copiar_nombre_codigo
* copiar_tabla_registro
* eliminar_entidades_area_pequeña
* sustituir_o_eliminar_entidades_por_area
* sustituye_area_por_puntual
* cortar
* cortar_poligono
* linea_a_poligono
* partir_poligonos
* recortar_poligono
* recortar_poligonos
* tram_inserta_vertices_entidad_sel
* detectar_lineas_perimetro_inferior
* detectar_undershoot
* unir_poligonos
